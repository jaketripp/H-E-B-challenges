## Developer II Challenge

### Instructions

```
cd dev_2_challenge
npm install
```
Now start the API server:
```
node server/server.js
```

Open a new terminal and navigate back to ```dev_2_challenge```

Build everything:
```
npm run build
cd build
http-server -p 3000
```
Now go to [localhost:3000](http://localhost:3000).

Enjoy!
